Chi tiết từng action:

📋 Duplicate Course:

Mở modal "Create Course" với data copy từ course gốc
Title: "Duplicate Course: IELTS 6.5 - Morning Class"
Tên course tự động thêm " (Copy)" ở cuối
Ngày bắt đầu/kết thúc reset về trống
Students không copy theo
👥 Manage Students:

Chuyển đến trang chi tiết course
Tự động active tab "Students"
Focus vào button "Add Student"
📊 View Statistics:

Chuyển đến trang chi tiết course
Tự động active tab "Stats"

5. LOADING STATES
Khi click View Details:

Card có loading overlay
Skeleton screen cho detail page
Breadcrumb loading
Khi submit Edit:

Button loading spinner
Form disabled
Success toast sau khi update
Khi delete:

Confirmation modal loading
Success redirect về courses list
6. ERROR STATES
Edit form validation:

Required fields highlight đỏ
Error messages dưới inputs
Submit button disabled until valid
Delete errors:

Network error: Retry button
Permission error: Contact admin message
Navigation errors:

404 course: Redirect về courses list
Permission denied: Show error page
7. RESPONSIVE BEHAVIOR
Mobile (<768px):

Detail page: Full screen
Edit modal: Full screen
More menu: Bottom sheet style
Delete confirmation: Full width
Tablet (768-1023px):

Detail page: Full width
Modals: 90% width, centered
Dropdown: Adjust position if near edge
